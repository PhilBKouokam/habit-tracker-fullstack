# Habit Tracker App

A full-stack application to track daily habits.

## Features
- Add habits
- Mark as complete/uncomplete
- Delete habits
- Persistent storage with MongoDB
- Loading + error + success feedback

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Bootstrap (UI)

## How to run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
